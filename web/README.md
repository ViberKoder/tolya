# Jetton Minter Web Interface

Веб-интерфейс для создания Jetton 2.0 токенов в сети TON.

## Установка

```bash
cd web
npm install
```

## Запуск

```bash
npm start
```

Приложение будет доступно по адресу http://localhost:3000

## Сборка для продакшена

```bash
npm run build
```

## Особенности

- Интеграция с TON Connect для подключения кошелька
- Современный дизайн в стиле ton.org
- Поддержка mainnet и testnet
- Простой и интуитивный интерфейс

## Технологии

- React 18
- TypeScript
- TON Connect UI
- @ton/core для работы с блокчейном
